# src目录结构及文件

├── module ------------------------------- 提供对 module 的处理，最后构建成一棵 module tree
├── plugins ------------------------------- devtool插件和logger插件所在目录
│   ├── git-hooks ------------------------- 存放git钩子的目录
├── helpers.js ------------------------------- 存放辅助函数的文件，例如mapMutations这样的api
├── index.js&index.cjs.js ------------------------------- 入口文件，一个用于 commonjs的打包、一个用于esmodule的打包
├── mixin ------------------------------- 提供 install 方法，用于注入 $store
├── store.js ------------------------------- vuex 的核心代码
├── util ------------------------------- 工具函数文件
